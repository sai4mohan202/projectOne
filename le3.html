Deployment Instructions

Follow these steps to deploy your StockIndicator Pro application:

1. Local Development Setup

git clone https://github.com/stockindicatorpro/webapp.git
cd webapp
npm install

2. Configure Environment Variables

Create a .env file with your API keys and configuration settings.

API_KEY=your_api_key
MARKET_DATA_URL=https://api.marketdata.com
ENABLE_3D=true

3. Run Development Server

npm run dev

This will start the development server at http://localhost:3000

4. Build and Deploy

npm run build
npm run deploy

This will build your application and deploy it to your configured hosting provider.

For more detailed instructions and troubleshooting, please refer to our full documentation.
